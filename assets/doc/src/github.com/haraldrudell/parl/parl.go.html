     6  /*
     7  Package parl handles inter-thread communication and controls parallelism
     8  
     9  Logging of extected output is via Out(string, ...interface{}).
    10  Parl logging uses comma separator for numbers and is thread safe
    11  
    12  Log(string, ...interface{}) always outputs to stderr.
    13  Console is the same intended to be used for command-line interactivity.
    14  SetDebug(true) appends code location
    15  
    16  Info is active by default and outputs to stderr.
    17  SetSilent(true) removes this output.
    18  SetDebug(true) appends code location
    19  IsSilent deteremines if Info printing applies
    20  
    21  Debug only prints if SetDebug(true) or the code location matches SetInfoRegexp().
    22  The string matched for regular expression looks like: “github.com/haraldrudell/parl.FuncName”
    23  IsThisDebug determines if debug is active for the executing function
    24  
    25  parl.D is intended for temporary printouts to be removed before check-in
    26  
    27  parl provides generic recovery for goroutines and functions:
    28  capturing panics, annotating and storing errors and invoking an error handling function on errors:
    29  
    30    func f() (err error) {
    31  		defer parl.Recover(parl.Annotation(), &err, onError func(e error) { … })
    32  	…
    33  Default error string: “Recover from panic in somePackage.someFunction: 'File not found'.
    34  For multiple errors, Recover uses error116 error lists,
    35  while Recover2 instead invokes onError multiple times
    36  
    37  Parl is about 9,000 lines of Go code with first line written on November 21, 2018
    38  
    39  On 3/16/2022 Parl was open-sourced under an ISC License
    40  
    41  © 2020–present Harald Rudell <harald.rudell@gmail.com> (https://haraldrudell.github.io/haraldrudell/)
    42  */
